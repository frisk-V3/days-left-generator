# 残り〇〇日ジェネレーター
## 指定した目標日までの「残り日数」を表示するためのテキスト、HTMLタグ、画像を生成するツールです。
### 概要
- このリポジトリには、Webブラウザで動く「HTML/JS版」と、Windowsで動作する「C#版」の2種類のジェネレーターが含まれています。
### 使い方
### Web版
- index.html をブラウザで開きます。
- 目標の日付とイベント名を入力します。
- 生成されたテキストやHTMLコードをコピーして利用してください。
### C#版（Windowsデスクトップ）
- Program.exe を起動します。
-日付を選択し、「画像として出力」をクリックします。
- 実行ファイルと同じフォルダに generated_days.png が生成されます。
